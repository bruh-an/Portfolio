IAT 339 P03 Process Style Guide

In-text link

Note: I do not use interactable text links for my portfolio, they are typically paired with buttons (as seen in astrohno.html)

CSS selector:

a, link

Sample code:

<a class="link" href="https://canvas.sfu.ca/">A link to the Canvas site</a>

Rendered element:

A link to the Canvas site

A navigation link

CSS selector:

nav, navigation, a

Sample code:

<nav> <div class="navigation"> <a href="index.html">Default Navigation</a> <div> </nav>

Rendered element:

Default Navigation

Buttons

Note: The first button was not used

Note 2: The third button is interactable and is seen in my portfolio

CSS selector:

.button, a

Sample code:

<button class="button">Redirect</button> <a class="button">Redirect</a> <a class="button" href="https://canvas.sfu.ca/">To Canvas</a>

Rendered elements:

Redirect To Canvas

Headings

Note: Heading styling (such as margins/spacing) vary depending on their parent container

CSS selector:

h1, h2, h3, h4

Sample code:

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <h4>Heading 4</h4>

Rendered element:

Heading 1

Heading 2

Heading 3

Heading 4

Paragraphs

Note: Paragraph spacing and margins vary depending on their parent container

CSS selector:

p

Sample code:

<p> This is a paragraph. </p>

Rendered element:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Donec a diam lectus. Sed sit amet ipsum mauris. Maecenas congue ligula ac quam viverra nec consectetur ante hendrerit. Donec et mollis dolor.

Numbered Lists and Unordered Lists

Note: Numbered Lists are not used and unordered lists are only used for the citation page

CSS selector:

ol, ul, li

Sample code:

<ol> <li>First item</li> <li>Second item</li> <li>Third item</li> </ol> <ul> <li>Hello</li> <li>And</li> <li>Goodbye</li> </ul>

Rendered element:

  1. First item
  2. Second item
  3. Third item
  • Hello
  • And
  • Goodbye

Main Navigation

Note: Navigation bar is visible at the top. The box background (navigation-wrapper) can not be displayed here

a, nav, navigation, navigation-wrapper

Sample code:

<nav> <div class="navigation"> <a href="#">HOME</a> <a href="#">SnapBudget</a> <a href="#">AstrOh-No!</a> </div> </nav>

HOME SnapBudget AstrOh-No!

Banners

Note: Banners look much better on project pages themselves where they span the whole page width

CSS selector:

.banner-area, .banner-area-2, .banner-area figure, .banner-area-2 figure, .center-h2-with-underline, a, img, figcaption

Sample code:

<div id="banner class="banner-area"> <h2 class="center-h2-with-underline">Banner Heading</h2> <figure> <img src="#" alt"just an image that does not exist"> <figcaption>here is a figcaption</figcaption> </figure> </div>

Rendered element:

Banner Heading

Fonts

Note: Although h3 and navigation a use the same font, their text styling is different

CSS selector:

h1, h2, h3, p, body, figcaption, .button, navigation a

Sample code:

<h2>Afacad</h2> <h3>Asap<h3> <p>Assistant</p>

Rendered element:

h1, h2, button: Afacad

h3, navigation a: Assistant

body, p, figcaption: Asap

Colour Palettes

CSS selector:

body, nav, footer

Sample colors:

Common colors: background-color: #171717; (for navigation-wrapper, footer, banner-area). color: #00a6ff; (blue for several elements to provide colour contrast). background-color: #015581; (border color for navigation-wrapper and footer). Text (h1, h2, p, etc.) colors mostly use white (#FFFFFF) or anything close such as #F1F1F1 for body. They may be swapped often for the color #00a6ff.
